Twisted layout

Layout has one stretch side (left) and one static (right). Good point is to set equal height to both sides. Container must have two classes: .twisted-container-<threshold> and .twisted-container-<width>. Following threshold classes are supported:

  • .twisted-container-xs
  • .twisted-container-sm
  • .twisted-container-md
  • .twisted-container-lg

Following width classes are supported:

  • .twisted-container-2 (side width=16.66666667%)
  • .twisted-container-4 (side width=33.33333333%)
  • .twisted-container-6 (side width=50%)
  • .twisted-container-8 (side width=66.66666667%)
  • .twisted-container-10 (side width=83.33333333%)
